# Poker Room Story v0.8.5

Patch: table route flow, five real room tables, readable revealed cards.

## Install

Replace the contents of `C:\PokerRoomStory` with this archive, then run:

```powershell
cd C:\PokerRoomStory
python -m http.server 8080
```

Open `http://localhost:8080`.

## Git push

```powershell
cd C:\PokerRoomStory
git add .
git commit -m "Patch v0.8.5 table route flow"
git push
```

After GitHub Pages deploy: `Проверить → Принудительно обновить → Ctrl + F5`.
