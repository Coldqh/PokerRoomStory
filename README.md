# Poker Room Story v0.4.2

## Patch v0.4.2 — Mobile Nav / Table Pace

Маленький аккуратный патч без крупных систем.

### Главное

- убрана некрасивaя горизонтальная полоса у нижней навигации на маленьких экранах;
- навигация теперь сжимается без внутреннего scroll-bar;
- на очень узких экранах нижняя панель показывает только иконки;
- добавлен переключатель темпа анимаций: `Обычный → Быстрый → Мгновенный`;
- темп сохраняется в сейве;
- исправлен мелкий риск в `screens.js`: убран дублирующий `const result`;
- fold guard из v0.4.1 сохранён;
- версия обновлена до `0.4.2` во всех местах;
- service worker cache обновлён до `0.4.2`.
