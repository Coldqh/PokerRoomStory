# Poker Room Story v0.4.1

## Patch v0.4.1 — Fold Guard

Маленький технический патч. UI-стиль и большие системы не трогались.

### Главное

- исправлен баг: игрок/NPC после `Fold` больше не может получить ход на следующих улицах;
- очередь действий теперь дополнительно пропускает `folded` и `all-in` места;
- `currentActorId` больше не может указывать на сброшенного игрока;
- folded-seat визуально стал тише: вместо карт показывается компактная метка `Fold`;
- версия обновлена до `0.4.1` во всех местах: `appMeta`, `sw.js`, `version.json`, `index.html`, content pack;
- service worker cache обновлён до `0.4.1`.

### Проверка

Проверено 200 тестовых раздач:

- раздачи доходят до результата;
- folded игроки не получают новых действий;
- текущий актор не становится folded/all-in игроком;
- базовая загрузка модулей не падает.
