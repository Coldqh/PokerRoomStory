# Poker Room Story v0.1.5

Static mobile-first poker room prototype.

## Patch v0.1.5

- cleaner minimal interface;
- less text noise;
- calmer club screens;
- focused table screen;
- compact action feed;
- better mobile spacing;
- short hand hints and event messages.

## Run locally

```bash
python -m http.server 8080
```

Open:

```text
http://localhost:8080
```
