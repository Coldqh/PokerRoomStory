# Poker Room Story v0.4.3

## Patch v0.4.3 — Timeline Fold Fix

Технический фикс без новых крупных систем.

### Главное

- исправлена причина, из-за которой folded-игроки визуально появлялись в фолде слишком рано;
- анимация теперь использует snapshot состояния на каждом событии, а не финальное состояние всей раздачи;
- folded NPC больше не должен выглядеть активным в следующих улицах;
- старые активные раздачи из версий до `0.4.3` безопасно сбрасываются при загрузке;
- банкролл, карьера, коллекции и прогресс не сбрасываются;
- текущий актор дополнительно проверяется: если он folded/all-in из старого сейва, активная рука сбрасывается;
- версия обновлена до `0.4.3`;
- service worker cache обновлён до `0.4.3`.

### Что не менялось

- новых больших систем нет;
- save schema прежняя: `2`;
- UI-стиль не трогался;
- баланс NPC почти не трогался.
