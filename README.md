# Poker Room Story v0.5.1

Patch: **Tasks Board**.

## Что нового

- Добавлена отдельная вкладка **Задания**.
- Активных заданий максимум **6**.
- Выполненное задание уходит в блок **Выполненные**.
- На место выполненного автоматически появляется следующее задание из пула.
- Награды стали понятнее: XP / Rep показываются на карточке и в журнале.
- Челленджи теперь имеют сложность: лёгкое / среднее / сложное / редкое.
- Добавлены новые типы заданий:
  - собрать стрит;
  - собрать флеш конкретной масти;
  - выиграть крупный банк;
  - выиграть showdown с сильной рукой;
  - сыграть серию рук;
  - сделать несколько fold.

## Установка

Распаковать архив с заменой файлов в:

```powershell
C:\PokerRoomStory
```

Запуск локально:

```powershell
cd C:\PokerRoomStory
python -m http.server 8080
```
