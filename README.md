# Poker Room Story v0.4.4

## Patch v0.4.4 — Hard Fold State Fix

Технический фикс без новых крупных систем.

### Главное

- добавлен жёсткий guard: folded/all-in игрок не может выполнить действие;
- currentActorId автоматически очищается, если указывает на folded/all-in игрока;
- очищены stale-события после анимации, когда ход переходит к игроку;
- folded NPC визуально выключен сильнее: без ставки и лишнего active-состояния;
- старые активные раздачи из версий до `0.4.4` безопасно сбрасываются при загрузке;
- банкролл, карьера, коллекции и прогресс не сбрасываются;
- текущий актор дополнительно проверяется: если он folded/all-in из старого сейва, активная рука сбрасывается;
- версия обновлена до `0.4.4`;
- service worker cache обновлён до `0.4.4`.

### Что не менялось

- новых больших систем нет;
- save schema прежняя: `2`;
- UI-стиль не трогался;
- баланс NPC почти не трогался.
